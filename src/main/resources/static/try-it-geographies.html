View geographies

The EDS Geography standard is the Geography, Market, Region and Country Unit structure based on the IBM management system alignment. IBM reports its financial performance using this structure within EPM.

However, outside of EPM, the Country Unit does not have as much meaning. Financial, HR, fulfillment, and workforce management applications generally operate at either the financial or ISO country (level 00 in the EDS).

The BMS Geography taxonomies.

Scenario information

  • Category: RESTful Data Retrieval
  • APIs: Data retrieval APIs run against the EDS and BMS geographic taxonomies and returning the data within a JSON tree.

Geographies

Nodes within a taxonomy often need to be aggregated in multiple ways depending on the perspective of the information gatherer looking at the node's detail. Alternative taxonomies provide a way to have multiple perspectives on the same data.

This "try-it" page shows different sets of Geography taxonomies and returns hierarchically organized JSON data visualized within a navigable D3 tree.
